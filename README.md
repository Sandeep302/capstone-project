# capstone-project
Project Title: SkyScan Live
Project Goal: A real-time dashboard that fetches and displays weather metrics and Air Quality Index (AQI) for a user-specified city.

Core Features (MVP):

Connect to a weather API to retrieve live temperature data.

Simple CLI interface to input city names.

Display basic "Feels Like" and humidity levels.

Future Features:

Real-time Alerts: Desktop notifications for severe weather warnings.

Data Logging: Saving daily snapshots to a local SQLite database for historical analysis.

GUI Upgrade: Moving from a command line to a web dashboard using Streamlit.
